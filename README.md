Role-Based Dictionary System using Val Tree (C++)
Description

This project implements a dictionary application using a Val Tree data structure in C++ with role-based access control.
Admins have full access to add, delete, search, search similar words, and display all entries. Users can only search and display words.
It demonstrates efficient data storage and retrieval using trees and basic role-based permissions.

Features
Admin Role:

Add words

Delete words

Search words

Search similar words

Display dictionary

User Role:

Search words

Display dictionary

Key Highlights:

Efficient word storage using Val Tree

Role-based access control for Admin and User

Written in C++ for easy compilation and execution

How to Compile and Run:

Open terminal/command prompt

Navigate to the project folder:

cd path/to/project


Compile the code:

g++ main.cpp -o dictionary


Run the program:

./dictionary       # Linux/Mac
dictionary.exe     # Windows


Login as Admin or User and perform actions based on role.
